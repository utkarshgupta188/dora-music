# Dora Player - Music Streaming App

A modern web-based music player application built with Flask and JavaScript.

## Features

- Search for songs
- Play music with a modern audio player interface
- Get song recommendations
- Download songs
- Modern and responsive UI
- Audio visualizations

## Technologies Used

- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- External API Integration
- Modern CSS Features (Grid, Flexbox, CSS Variables)

## Setup

1. Clone the repository:
```bash
git clone <your-repository-url>
cd music_app
```

2. Create a virtual environment and activate it:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python app.py
```

5. Open your browser and visit `http://localhost:5000`

## Usage

1. Search for songs using the search bar
2. Click on any song to play
3. Use the player controls at the bottom to:
   - Play/Pause
   - Skip to next/previous track
   - Adjust volume
   - See track progress
4. Click "Similar" to get song recommendations
5. Click "Download" to download a song
